# Job Tracker App

## Description
A full-stack web application that helps job seekers track their job applications, manage application statuses, and never miss a follow-up opportunity.

## Tech Stack
- **Frontend:** Next.js (TypeScript, Tailwind CSS)
- **Backend:** Spring Boot (Java)
- **Database:** PostgreSQL
- **Authentication:** JWT with refresh tokens

## Features
- [ ] Track job applications with company details, role, and status
- [ ] Dashboard with summarized progress overview
- [ ] Automatic status transitions (Submitted → Waiting → Ghosted)
- [ ] Secure storage for company portal credentials
- [ ] Reapply notifications after 6 months

## Project Status
Currently in development — Phase 4: Implementation

## Getting Started

### Prerequisites
- Java 21
- Node.js 20+
- PostgreSQL 17

### Run Backend
cd backend
./mvnw spring-boot:run

### Run Frontend
cd frontend
npm install
npm run dev
